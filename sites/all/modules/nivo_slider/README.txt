###   ABOUT 

Nivo Slider, Version 1.0
Authors: n8tron and glerous
Requirements: Drupal 6.x, Views, jQuery update

###   INSTALLATION 

1. Download the production version of the Nivo Slider from http://nivo.dev7studios.com/
   and place it in your sites/all/libraries folder with the name nivoslider.

2. Download and unzip the Nivo Slider module into your modules directory.

3. Goto Administer > Site Building > Modules and enable Nivo Slider.

4. Add views template to theme views-view-fields--nivo-slider.tpl.php

5. Goto Administer > Site Building > Views and edit or add a view.

6. In the style options choose Nivo Slider.

7. Optionally, Select the settings icon next to the Nivo Slider style and choose custom
   settings for the view.

###   DEPENDENCIES

- Libraries API Module

###   NOTES

- Uses the jQuery Nivo Slider plugin http://nivo.dev7studios.com/
